# workflow-orchestration

This repository is my personal workflow operating system for planning, executing, and improving complex tasks.

## Structure
- `AGENTS.md` — Core rules and orchestration principles
- `tasks/todo.md` — Active tasks and planning
- `tasks/lessons.md` — Mistakes, learnings, and process improvements

## How I Use This
1. Add a real task to `tasks/todo.md`
2. Follow the rules in `AGENTS.md`
3. Execute the task
4. Log mistakes and improvements in `tasks/lessons.md`
